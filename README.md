# AI-Resume-Screening-Job-Recommendation-System
An intelligent recruitment assistance system that uses AI, NLP, and Machine Learning to analyze resumes, evaluate candidate profiles, and recommend suitable job roles.

Features

1.PDF resume text extraction
2.Skill, qualification, and experience analysis
3.Required & additional requirement matching
4.Resume match score using ML
5.Job role and company recommendations

Technologies Used

1.Python
2.PyPDF2
3.Regular Expressions
4.Scikit-learn
5.TF-IDF & Cosine Similarity

Modules 

1.Resume Text Extraction
2.Skill & Qualification Analysis
3.Experience Detection
4.Machine Learning Scoring
5.Job Recommendation

How to Run

1.Clone the repository
2.Install required libraries
3.Place resume PDF in project folder
4.Run the Python script
